# pictures-classify
现有一个文件夹A，里边存放有20种信噪比的图片，每种信噪比下包含11种类型的图片，图片的命名可以说明图片的种类和信噪比（信噪比按字符顺序排放）。
read1:#将一个文件夹下的混乱的cucuncut中的图片按照信噪比分开，每个信噪比的图片保存为一个文件夹。
test-train2: #从分开信噪比的图片中，按照一定的比例，划分出测试集，分别存为一个文件夹。即得到20个训练文件夹，20个测试文件夹。
mix-snrs3:#针对按信噪比分别存放图片的多个训练文件夹（例如20个文件夹）。将20个信噪比的图片，按照输入的信噪比范围揉和成训练集

mixtestsnr2testevrsnr4:#将分开的20个测试集文件夹揉合为一个文件夹.

我的执行顺序：read1》》test-train2》》mix-snrs3》》mixtestsnr2testevrsnr4
